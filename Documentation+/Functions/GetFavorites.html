GetFavorites

Returns the list of the favorites with the given type. Возвращает Список избранного выбранного типа элемента(-ов). 

def GetFavorites (typeID) -> List (String)

 

Parameters

typeID
[in] Element type ID. See API_ElemTypeID for available element types.
[in] Идентификатора типа элемента. Посмотреть API_ElemTypeID для доступных типов элементов.
variationID (optional)
[in] Element variation ID, optional parameter. See API_ElemVariationID for available element variations.
[in] ID вариации элемента, необязательный параметр. Посмотреть API_ElemVariationID доступные вариации для элемента. (Большинство уже не используется)

 

Return Values

This function returns the list of the names of favorites which have the given type.
Эта функция возвращает список имен избранных, которые имеют данный тип.

 

Remarks

This function is analogous to the ACAPI_Favorite_GetNum function of the C++ API.
Эта функция является аналогом ACAPI_Favorite_GetNum функции в C++ API.

 

Example

wallFavoriteList = GetFavorites(API_WallID)
print("Number of favorites with wall type: {}".format(len(wallFavoriteList)))

 

Requirements

Version: Python API 0.2 or Later
Module: GS
