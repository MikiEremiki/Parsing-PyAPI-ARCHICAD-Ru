GetPropertyDefinitionDictionary

(en)Lists the Property Definitions in a dictionary.
(ru)Перечисляет определения свойств в словаре. 

def GetPropertyDefinitionDictionary (groupGUID) -> Dict (String, APIObject)

 

Parameters

groupGUID
[in] (en)GUID of a Property Group.
[in] (ru)GUID группы свойств.

 

Return Values

(en)A dictionary containing the available Property Definitions belong to the given Property Group. Keys of the items are the GUIDs of the property definitions.
(ru)Cловарь, содержащий доступные определения свойств, относится к данной группе свойств. Ключи элементов являются идентификаторами GUID определений свойств.

 

Remarks

(en)This function is analogous to the ACAPI_Property_GetPropertyDefinitions function of the C++ API.
(ru)Эта функция аналогична функции ACAPI_Property_GetPropertyDefinitions из C++ API.

(en)For information on how to interpret the returned APIObject dictionary values, visit the C++ documentation.
(ru)Для получения информации о том, как интерпретировать возвращенные значения словаря APIObject, посетите раздел C++ documentation.

 

Example

for groupGuid in GetPropertyGroupDictionary().keys():
    for definition in GetPropertyDefinitionDictionary(groupGuid).values():
        print("{} {}".format (definition.name, definition.guid))
