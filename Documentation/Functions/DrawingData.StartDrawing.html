StartDrawing

(en)Enters Drawing mode for a DrawingData structure.
(ru)Входит в режим рисования для структуры DrawingData. 

def StartDrawing () -> None

 

Remarks

(en)Entering Drawing mode makes each subsequent call to any Element creation function to be redirected to the Drawing Data, which means no actual Element creation.
(ru)При входе в режим рисования каждый последующий вызов любой функции создающей элемент, будет перенаправлена в данные чертежа, что означает отсутствие фактического создания элемента.

(en)Only one DrawingData can be in Drawing mode at one time, as this is a global state. Subsequent calls to StartDrawing, or using StartDrawing and StopDrawing on two different instances of DrawingData results in undefined behavior.
(ru)Только один DrawingData может быть в режиме рисования в одно время, так как это глобальное состояние. Последующие вызовы для StartDrawing, или использование StartDrawing и StopDrawing на двух различных экземплярах DrawingData приводит к неопределенному поведению.
