GetElementList

(en)Returns a list of GUIDs of the elements of the given type.
(ru)Возвращает список всех GUID элементов данного/переданного/выбранного типа. 

def GetElementList (typeID) -> List (String)

 

Parameters

typeID
[in] (en)Element type ID. See API_ElemTypeID for available element types.
[in] (ru)Идентификатора типа элемента. Посмотреть API_ElemTypeID для доступных типов элементов.

 

Return Values

(en)This function returns the list of the GUIDs of elements which have the given type.
(ru)Эта функция возвращает список всех GUID элементов данного/переданного/выбранного типа.

 

Remarks

(en)This function is analogous to the ACAPI_Element_GetElemList function of the C++ API.
(ru)Эта функция является аналогом ACAPI_Element_GetElemList функции в C++ API.

 

Example

wallList = GetElementList(API_WallID)
print("Number of walls: {}".format(len(wallList)))

 

Requirements

Version: Python API 0.1 or Later
Module: GS
