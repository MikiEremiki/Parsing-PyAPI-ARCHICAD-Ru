PublishSet

(en)Publishes the given publisher set to the given (optional) path.
(ru)Публикует заданный издателем набор по заданному (необязательному) пути. 

def PublishSet (setName, path) -> None

 

Parameters

setName
[in] (en)The name of an existing publisher set.
[in] (ru)Имя существующего набора издателей.
path
[in] (en)(optional) The function will publish to this path.
[in] (ru)(необязательно) Функция опубликует этот путь.

 

Return Values

(en)This function does not have any return values.
(ru)Эта функция не имеет возвращаемых значений.

 

Remarks

(en)If the path parameter is not set then the default path of the publisher set will be used.
(ru)Если параметр path не задан, то будет использоваться путь по умолчанию для набора издателей.

(en)This function is analogous to the APIDo_PublishID function of the C++ API.
(ru)Эта функция аналогична функции APIDo_PublishID из C++ API.

 

Example

import os
import sys

scriptDir = os.path.dirname (sys.argv[0])
PublishSet ("1 - Views", scriptDir)
