GetZone

(en)Queries an Zone element.
(ru)Запрашивает элемент зоны. 

def GetZone (guid) -> APIObject

 

Parameters

guid
[in] (en)A string containing the GUID of the Zone.
[in] (ru)Строка, содержащая идентификатор GUID зоны.

 

Return Values

(en)This function returns an APIObject describing the queried Zone element.
(ru)Эта функция возвращает объект APIObject, описывающий запрашиваемый элемент зоны.

 

Remarks

(en)This function is analogous to the ACAPI_Element_Get function of the C++ API for Zone elements.
(ru)Эта функция аналогична функции ACAPI_Element_Get из C++ API для элементов зоны.

(en)For information on how to interpret the passed APIObject, visit the C++ documentation.
(ru)Для получения информации о том, как интерпретировать переданный APIObject, посетите раздел C++ documentation.
