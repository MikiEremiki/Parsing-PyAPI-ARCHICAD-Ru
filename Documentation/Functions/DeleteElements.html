DeleteElements

(en)Deletes a list of elements.
(ru)Удаляет список элементов. 

def DeleteElements (elemGuidList) -> Integer

 

Parameters

elemGuidList
[in] (en)List of element GUIDs.
[in] (ru)Список GUID элементов.

 

Return Values

(en)This function returns the count of the deleted elements.
(ru)Эта функция возвращает количество удаленных элементов.

 

Remarks

(en)This function is analogous to the ACAPI_Element_Delete function of the C++ API.
(ru)Эта функция аналогична функции ACAPI_Element_Delete из C++ API.

 

Example

DeleteElements (GetSelectedElements ())
