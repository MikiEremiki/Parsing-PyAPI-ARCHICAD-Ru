CreateElementGroup

(en)Creates a new group from the given elements.
(ru)Создает новую группу из заданных элементов. 

def CreateElementGroup (elemGuidList) -> String

 

Parameters

elemGuidList
[in] (en)List of element GUIDs.
[in] (ru)Cписок GUID элементов.

 

Return Values

(en)This function returns the GUID of the created Group.
(ru)Эта функция возвращает идентификатор GUID созданной группы.

 

Remarks

(en)This function is analogous to the ACAPI_ElementGroup_Create function oа the C++ API.
(ru)Эта функция аналогична функции ACAPI_ElementGroup_Create из C++ API.

 

Example

# Group morphs with similar material
morphDict = dict()

for morphGuid in GetElementList(API_MorphID):
    morph = GetElement(morphGuid)
    material = morph[0].material.attributeIndex
    if material not in morphDict:
        morphDict[material] = list()
    morphDict[material].append(morphGuid)

with UndoScope("Create Groups for Morphs"):
    for similarMorphs in morphDict.values():
        CreateElementGroup(similarMorphs)

print("{} Groups Created".format (len (morphDict.values())))
