GetArc

Queries an Arc element. 

def GetArc (guid) -> APIObject

 

Parameters

guid
[in] A string containing the GUID of the Arc.

 

Return Values

This function returns an APIObject describing the queried Arc element.

 

Remarks

This function is analogous to the ACAPI_Element_Get function of the C++ API for Arc elements.

For information on how to interpret the passed APIObject, visit the C++ documentation.

 

Requirements

Version: Python API 0.1 or Later
Module: GS
