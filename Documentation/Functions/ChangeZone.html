ChangeZone

(en)Modifies a Zone element.
(ru)Изменяет элемент зоны. 

def ChangeZone (element, memo) -> String

 

Parameters

element
[in] (en)An APIObject describing the element to modify.
[in] (ru)APIObject, описывающий элемент для изменения.
memo
[in] (en)An APIObject containing the zone parameters. Only addPars is used.
[in] (ru)APIObject, содержащий параметры зоны. Только addPars используется.

 

Return Values

(en)This function returns the GUID of the modified element.
(ru)Эта функция возвращает идентификатор GUID измененного элемента.

 

Remarks

(en)This function is analogous to the ACAPI_Element_Change function of the C++ API for Zone elements. By setting a member of the passed APIObject you also mask it for the operation.
(ru)Эта функция аналогична функции ACAPI_Element_Change из C++ API для элементов зоны. Путем установки элемента пройденного APIObject вы также маскируете его для операции.

(en)For information on how to fill the passed APIObject, visit the C++ documentation.
(ru)Для получения информации о том, как заполнить переданный APIObject, посетите раздел C++ documentation.

 

Example


def ChangeZoneNumber (guid):
    element = GetZone (guid)
    count = 1
    element[0].roomNoStr = 'NEW No {}'.format(count)
    return ChangeZone (element, APIObject ())

for elemGuid in GetSelectedElements ():
    element = GetElement (elemGuid)
    typeID = GetElementType (elemGuid)
    if (typeID == API_ZoneID):
        ChangeZoneNumber (element)
