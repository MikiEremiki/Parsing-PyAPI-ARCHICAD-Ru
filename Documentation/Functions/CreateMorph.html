CreateMorph

(en)Creates an Morph element.
(ru)Создает элемент Morph. 

def CreateMorph (element) -> String

 

Parameters

element
[in] (en)An APIObject describing the Morph element to create.
[in] (ru)APIObject, описывающий элемент Morph для создания.

 

Return Values

(en)This function returns the GUID of the created element.
(ru)Эта функция возвращает идентификатор GUID созданного элемента.

 

Remarks

(en)This function is analogous to the ACAPI_Element_Create function of the C++ API for Morph elements. Undefined members will be set to the current defaults.
(ru)Эта функция аналогична функции ACAPI_Element_Create из C++ API для элементов Morph. Неопределенные члены будут установлены на текущие значения по умолчанию.

(en)For information on how to fill the passed APIObject, visit the C++ documentation.
(ru)Для получения информации о том, как заполнить переданный APIObject, посетите раздел C++ documentation.

 

Example

See DrawChessBoard.py and DeclareOBJImport.py example scripts.
