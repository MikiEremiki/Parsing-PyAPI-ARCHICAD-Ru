GetPlanLocation

Returns the location of the current file in the file system. 

def GetPlanLocation () -> String

 

Return Values

This function returns the path of the current file, or None if the project hasn't been saved yet.

 

Remarks

This function returns the projectPath like in the APIEnv_GetProjectID function of the C++ API.

 

Example

print("Current plan path: {}".format (GetPlanLocation())

 

Requirements

Version: Python API 0.1 or Later
Module: GS
