Alert

(en)Displays an alert window with the given text.

def Alert (text) -> None

 

Parameters

text
[in] (en)A string to display in the alert window.
[in] (ru)Строка для отображения в окне предупреждения.

 

Return Values

(en)This function does not have any return values.
(ru)Эта функция не имеет возвращаемых значений.

 

Example
