CreateNavigatorItem

(en)Creates new navigator item with the given data under the given parent.
(ru)Создает новый элемент навигатора с заданными данными под заданным родительским элементом. 

def CreateNavigatorItem (item, parentItem, itemBefore) -> String

 

Parameters

item
[in-out] (en)An APIObject describing the data of the new navigator item. The identifier fields will be filled by this function.
[in-out] (ru)APIObject, описывающий данные нового элемента навигатора. Поля идентификаторов будут заполнены этой функцией.
parentItem
[in] (en)An existing navigator item which will be the parent of the new item (e.g. subset in case of layout items).
[in] (ru)Существующий элемент навигатора, который будет родительским для нового элемента (например, подмножество в случае элементов компоновки).
itemBefore
[in] (en)(optional) An existing navigator item to define the position of the new item.
[in] (ru)(необязательно) существующий элемент навигатора для определения положения нового элемента.

 

Return Values

(en)This function returns the GUID of the new navigator item.
(ru)Эта функция возвращает идентификатор GUID нового элемента навигатора.

 

Remarks

(en)For information on how to fill the passed item APIObject, visit the documentation of API_NavigatorItem structure.
(ru)Для получения информации о том, как заполнить переданный itemAPIObject, посетите раздел документации структуры API_NavigatorItem.
