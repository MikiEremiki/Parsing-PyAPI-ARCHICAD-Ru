StopDrawing

(en)Exits Drawing mode for a DrawingData structure, and initializes it.
(ru)Завершает работу в режиме рисования для структуры DrawingData и инициализирует ее. 

def StopDrawing () -> None

 

Remarks

(en)Exiting Drawing mode results in the DrawingData being initialized, if successful.
(ru)Выход из режима рисования приводит к инициализации DrawingData, если успешно.

(en)Only one DrawingData can be in Drawing mode at one time, as this is a global state. Subsequent calls to StopDrawing, or using StartDrawing and StopDrawing on two different instances of DrawingData results in undefined behavior.
(ru)Только один DrawingData может быть в режиме рисования в одно время, так как это глобальное состояние. Последующие вызовы для StopDrawing, или использование StartDrawing и StopDrawing на двух различных экземплярах DrawingData приводит к неопределенному поведению.

 

Requirements

Version: Python API 0.1 or Later
Module: GS
