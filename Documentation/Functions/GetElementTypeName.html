GetElementTypeName

Returns the localized name of a built-in element type.
Возвращает "локализованное имя" (а именно имя текстом с переводом на язык на который установленна программа, на 17.10.19 их 73) встроенного типа элемента. 

def GetElementTypeName (typeID) -> String

 

Parameters

typeID
[in] Element type identifier of an element. See API_ElemTypeID for available element types.
[in] Идентификатор типа элемента. Посмотреть список API_ElemTypeID для существующих типов элементов.

 

Return Values

This function returns the localized name of an element type.
Эта функция возвращает "локализованное имя" типа элемента.

 

Remarks

This function is analogous to the APIAny_GetElemTypeNameID function of the C++ API.
Эта функция является аналогом APIAny_GetElemTypeNameID функции в C++ API-интерфейс.

 

Example

print("Selected elements:")
for elemGuid in GetSelectedElements ():
    type = GetElementType(elemGuid)
    print("\t{}".format (GetElementTypeName(type)))
