GetNavigatorItem

(en)Retrieves a navigator item identified by it's GUID.
(ru)Извлекает элемент навигатора, определяемый его идентификатором GUID. 

def GetNavigatorItem (guid, mapId) -> APIObject

 

Parameters

guid
[in] (en)The GUID of the navigator item.
[in] (ru)Идентификатор GUID элемента навигатора.
mapId
[in] (en)(optional) ID of the navigator map. See API_NavigatorMapID for available navigator maps.
[in] (ru)(необязательный) Идентификатор карты навигатора. Посмотреть API_NavigatorMapID для доступных карт навигатора.

 

Return Values

(en)This function returns an APIObject which represents a navigator item.
(ru)Эта функция возвращает объект APIObject, который представляет элемент навигатора.

 

Remarks

(en)For information on how to interpret the returned APIObject, visit the documentation of API_NavigatorItem structure.
(ru)Для получения информации о том, как интерпретировать возвращенный APIObject, посетите страницу документация о API_NavigatorItem структуре .

 

Requirements

Version: Python API 0.2 or Later
Module: GS
