GetElementType

(en)Returns the type ID of a given element.
(ru)Возвращает идентификатор типа данного элемента. 

def GetElementType (elemGuid) -> Integer

 

Parameters

elemGuid
[in] (en)GUID of an element.
[in] (ru)GUID элемента.

 

Return Values

(en)This function returns the type ID of the given element. See API_ElemTypeID for available element types.
(ru)Эта функция возвращает идентификатор типа данного элемента. Видеть API_ElemTypeID для доступных типов элементов.

 

Remarks

(en)This function is analogous to the ACAPI_Element_GetHeader function of the C++ API.
(ru)Эта функция аналогична функции ACAPI_Element_GetHeader функция C++ API.

 

Example

print("Selected elements:")
for elemGuid in GetSelectedElements ():
    type = GetElementType(elemGuid)
    print("\t{}".format (GetElementTypeName(type)))
