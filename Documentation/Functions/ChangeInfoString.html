ChangeInfoString

Changes the info string (ID) of an element.
Изменяет информационную строку (ID) элемента. 

def ChangeInfoString (elemGuid, infoString) -> String

 

Parameters

elemGuid
[in] GUID of an element.
[in] Уникальный GUID элемента.
infoString
[in] New info string.
[in] Новая строка в поле ID.

 

Return Values

This function returns the GUID of the modified element.
Эта функция возвращает уникальный идентификатор измененного элемента.

 

Remarks

 

Example

itemNumber = 1

for elemGuid in GetSelectedElements ():
    infoString = 'NEW ID {}'.format (itemNumber)
    ChangeInfoString (elemGuid, infoString)
    itemNumber += 1
