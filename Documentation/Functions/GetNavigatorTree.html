GetNavigatorTree

(en)Returns the navigator items in a tree dictionary from the given navigator map.
(ru)Возвращает элементы навигатора в древовидном справочнике из данной карты навигатора. 

def GetNavigatorTree (mapID) -> Dictionary

 

Parameters

mapID
[in] (en)ID of the navigator map. See API_NavigatorMapID for available navigator maps.
[in] (ru)Идентификатор карты навигатора. Смотрите раздел API_NavigatorMapID для доступных карт навигатора.

 

Return Values

(en)This function returns the navigator items in a tree dictionary.
(ru)Эта функция возвращает элементы навигатора в древовидном словаре.

 

Example

def PrintNavigatorTree (tree, level):
    levelStr = ""
    for i in range(0,level):
        levelStr += "\t"
    for k in tree.keys():
        print(levelStr + k.name)
        PrintNavigatorTree (tree[k], level + 1)

tree = GetNavigatorTree(API_LayoutMap)
PrintNavigatorTree (tree, 0)
