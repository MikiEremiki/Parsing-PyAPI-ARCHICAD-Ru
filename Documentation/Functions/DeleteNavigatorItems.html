DeleteNavigatorItems

Creates new navigator item with the given data under the given parent. 

def DeleteNavigatorItems (itemList) -> Integer

 

Parameters

itemList
[in] List of APIObject objects describing existing navigator items.

 

Return Values

This function returns the count of the deleted items.

 

Remarks

 

Example

layoutTree = GetNavigatorTree (API_LayoutMap)

def GetItemsFromTree (tree, t):
    l = []
    for k in tree.keys():
        if k.itemType == t:
            l.append (k)
        l.extend (GetItemsFromTree (tree[k], t))
    return l

masters = GetItemsFromTree (layoutTree, API_MasterLayoutNavItem)
DeleteNavigatorItems (masters)
