ChangeArc

(en)Modifies an Arc element.
(ru)Изменяет элемент дуги. 

def ChangeArc (element) -> String

 

Parameters

element
[in] (en)An APIObject describing the Arc element to modify.
[in] (ru)APIObject, описывающий элемент дуги для изменения.

 

Return Values

(en)This function returns the GUID of the modified element.
(ru)Эта функция возвращает идентификатор GUID измененного элемента.

 

Remarks

(en)This function is analogous to the ACAPI_Element_Change function of the C++ API for Arc elements. By setting a member of the passed APIObject you also mask it for the operation.
(ru)Эта функция аналогична функции ACAPI_Element_Change из API C++ для элементов Arc. Путем установки элемента пройденного APIObject вы также маскируете его для операции.

(en)For information on how to fill the passed APIObject, visit the C++ documentation.
(ru)Для получения информации о том, как заполнить переданный APIObject, посетите раздел C++ documentation.
